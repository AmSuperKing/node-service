Bootstrap

Bootstrap 是最受欢迎的 HTML、CSS 和 JS 框架，用于开发响应式布局、移动设备优先的 WEB 项目。为所有开发者、所有应用场景而设计。Bootstrap 让前端开发更快速、简单。所有开发者都能快速上手、所有设备都可以适配、所有项目都适用。

本站素材源于Bootstrap官网，你可以继续在本站浏览相关文档，也可以前往Bootstrap 的官方网站——[Twitter Bootstrap]，在该站点上找到所有可用的文档和下载相关资源。

起步

简要介绍 Bootstrap，以及如何下载、使用，还有基本模版和案例，等等。

下载

Bootstrap （以版本 v3.x说明）提供以下几种方式帮助快速上手，每一种方式针对具有不同技能等级的开发者和不同的使用场景。可以根据适合自身需求的情况选择方式。

用于生产环境的 Bootstrap：

编译并压缩后的 CSS、JavaScript 和字体文件。不包含文档和源码文件。

下载Bootstrap

Bootstrap 源码:

Less、JavaScript 和 字体文件的源码，并且带有文档。需要 Less 编译器和一些设置工作。

下载源码

Sass:

这是 Bootstrap 从 Less 到 Sass 的源码移植项目，用于快速地在 Rails、Compass 或 只针对 Sass 的项目中引入。

BootstrapCDN引入

CDN引入，CDN的全称是Content Delivery Network，即内容分发网络。StackPath上的人们慷慨地为Bootstrap的CSS和JavaScript提供CDN支持。只需使用这些BootstrapCDN链接就可以引入Bootstrap到项目文件中。

<!-- 编译和压缩的 CSS 文件 -->
<link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/3.4.1/css/bootstrap.min.css" integrity="sha384-HSMxcRTRxnN+Bdg0JdbxYKrThecOKuH5zCYotlSAcp1+c8xmyTe9GYg1l9a69psu" crossorigin="anonymous">

通过Bower安装

您还可以使用Bower安装和管理Bootstrap的Less，CSS，JavaScript和字体：

xxxxxxxxxx
$ bower install bootstrap

通过npm安装

您也可以使用npm安装Bootstrap ：

xxxxxxxxxx
$ npm install bootstrap@3

require('bootstrap')将所有Bootstrap的jQuery插件加载到jQuery对象上。该bootstrap模块本身不出口任何东西。您可以通过/js/*.js在包的顶级目录下加载文件来分别手动加载Bootstrap的jQuery插件。

引导程序package.json在以下键下包含一些其他元数据：

  • less-Bootstrap的主要Less源文件的路径
  • style -使用默认设置（无自定义）预编译的Bootstrap非缩小CSS的路径

使用Composer安装

您还可以使用Composer安装和管理Bootstrap的Less，CSS，JavaScript和字体：

xxxxxxxxxx
$ composer require twbs/bootstrap

少/低要求的自动前缀

Bootstrap使用Autoprefixer处理CSS供应商前缀。如果要从其Less / Sass源代码编译Bootstrap而不使用Bootstrap自身的Gruntfile，则需要自己将Autoprefixer集成到构建过程中。如果您使用的是预编译的Bootstrap或Bootstrap自身的Gruntfile，则无需担心，因为Autoprefixer已集成到Bootstrap自身的Gruntfile中。

包含什么

Bootstrap有两种下载形式，您将在其中找到以下目录和文件，对公共资源进行逻辑分组，并提供已编译和最小化的变体。

需要jQuery

请注意，所有JavaScript插件都要求包含jQuery。点击查询，bower.json以了解支持哪些版本的jQuery。

预编译的引导程序

下载后，解压缩压缩文件夹以查看（已编译的）Bootstrap的结构。您会看到以下内容：

xxxxxxxxxx
bootstrap/
├── css/
│   ├── bootstrap.css
│   ├── bootstrap.css.map
│   ├── bootstrap.min.css
│   ├── bootstrap.min.css.map
│   ├── bootstrap-theme.css
│   ├── bootstrap-theme.css.map
│   ├── bootstrap-theme.min.css
│   └── bootstrap-theme.min.css.map
├── js/
│   ├── bootstrap.js
│   └── bootstrap.min.js
└── fonts/
    ├── glyphicons-halflings-regular.eot
    ├── glyphicons-halflings-regular.svg
    ├── glyphicons-halflings-regular.ttf
    ├── glyphicons-halflings-regular.woff
    └── glyphicons-halflings-regular.woff2

这是Bootstrap的最基本形式：预编译的文件，几乎可以在任何Web项目中快速使用。Bootstrap官方提供已编译的CSS和JS（bootstrap.*），以及已编译和精简的CSS和JS（bootstrap.min.*）。CSS源映射（bootstrap.*.map）可用于某些浏览器的开发人员工具。包括Glyphicons的字体，以及可选的Bootstrap主题。

引导程序源代码

Bootstrap源代码下载包括预编译的CSS，JavaScript和字体资源，以及源Less，JavaScript和文档。更具体地说，它包括以下内容和更多内容：

xxxxxxxxxx
bootstrap/
├── less/
├── js/
├── fonts/
├── dist/
│   ├── css/
│   ├── js/
│   └── fonts/
└── docs/
    └── examples/

bootstrap/下的less/，js/以及fonts/对应CSS，JS和图标字体（分别）的源代码。该dist/文件夹包括上面预编译下载部分中列出的所有内容。该docs/文件夹包含Bootstrap文档的源代码以及examples/Bootstrap用法。除此之外，任何其他包含的文件都提供对程序包，许可证信息和开发的支持。

基本模板

从下面基本的HTML模板开始，您可以自定义模板和示例，使它们适应您的需求。更多例子可以前往Bootstrap官方网站查看或者下载Bootstrap官方例子存储库

复制以下HTML以开始使用最小的Bootstrap文档。

xxxxxxxxxx
<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1">

​
    <!-- Bootstrap -->
    <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/3.4.1/css/bootstrap.min.css" integrity="sha384-HSMxcRTRxnN+Bdg0JdbxYKrThecOKuH5zCYotlSAcp1+c8xmyTe9GYg1l9a69psu" crossorigin="anonymous">
​
    <!-- HTML5 shim 和 Respond.js 是为了让 IE8 支持 HTML5 元素和媒体查询（media queries）功能 -->
    <!-- 注意: 通过 file:// 协议（就是直接将 html 页面拖拽到浏览器中）访问页面时 Respond.js 不起作用 -->
    <!--[if lt IE 9]>
      <script src="https://oss.maxcdn.com/html5shiv/3.7.3/html5shiv.min.js"></script>
      <script src="https://oss.maxcdn.com/respond/1.4.2/respond.min.js"></script>
    <![endif]-->
  </head>
  <body>
    <h1>Hello, world!</h1>
​
    <!-- jQuery (这是引导Bootstrap的 JavaScript插件所必需的) -->

    <!-- 加载 Bootstrap 的所有 JavaScript 插件。或者根据需要只加载单个插件 -->
